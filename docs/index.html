Companion is an image recognition library written with OpenCV. It’s supports an set of computer vision algorithms for CPU and Cuda usage. The source code is published on GitHub and is under GPLv3 license. Feel free if you want to implement additional algorithms from OpenCV to this library.

Supported Plattforms

Companion can be build for Windows, Linux and Universal Windows Platform. For example this library can be used as an UWP Project for a Microsoft Hololens.
