Constructor to create an CPU based feature matching algorithm.

Following feature matching can be used for example. Detector : BRISK, ORB, MSER, FastFeatureDetector, AgastFeatureDetector, GFTTDetector, SimpleBlobDetector, KAZE, AKAZE. Extractor : BRISK, ORB, KAZE, AKAZE. Matcher : FLANNBASED, BRUTEFORCE, BRUTEFORCE_L1, BruteForce-Hamming, BRUTEFORCE_HAMMINGLUT, BRUTEFORCE_SL2.

If you want to use SIFT: detector + descriptor and SURF: detector + descriptor you must build this lib with XFeatures2D support.

Parameters
detectorFeatureDetector to set.
extractorFeatureExtractor to set.
matcherFeatureMatcher to set.
matcherTypeFeatureMatcher type which is used like FlannBased or Bruteforce.
cornerDistanceHow many pixels the corners of a found area should be distant from each other. Default value is 10.
countMatchesHow much matches need to get an good matching result. Default is by 40.
useIRAIndicator to use IRA algorithm to use last detected objects from last scene. By default IRA is deactivated.
reprojThresholdHomography parameter: Maximum allowed reprojection error to treat a point pair as an inlier. Default is by 3.0.
ransacMaxItersHomography parameter: The maximum number of RANSAC iterations, 2000 is the maximum it can be. Default is by 500.
findHomographyMethodMethod used to computed a homography matrix. Default is by RANSAC.
